HyperText 1.0.0.0
Mirror: download.com

Were you ever sitting in a doctor's office without your laptop when you wanted to make a website? Have you realized that Dreamweaver isn't compatible with Windows Mobile? HyperText is an attempt at just that! While it may not have the advanced features of Dreamweaver, it will allow you to write code more easily and preview it using the mobile IE render engine without the need to save!

Features:

  • Insert menu to easily add tags to your code
  • Preview tab rendered using the mobile IE rendering engine
  • Remembers which files were recently worked on
  • Allows saving of files at HTML or can be used as a standard rich text editor
  • Runs on Windows mobile 5, 6, 6.1, and 6.5

Important:

  • This program is no longer supported.
  • The current version is the final release.
